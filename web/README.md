# Fluxor.io Website

React + TypeScript website for Fluxor framework built with Vite.

## Quick Start

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit http://localhost:3000

### Build

```bash
npm run build
```

Production files will be in `dist/` directory.

### Preview

```bash
npm run preview
```

## Components

- **Navbar** - Navigation bar with links
- **Hero** - Main hero section with code example
- **Features** - Feature highlights
- **Architecture** - Architecture overview
- **Comparison** - Comparison with other frameworks
- **Footer** - Footer with links

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS (via CDN)
- Lucide React (icons)

## Deployment

See [BUILD_INSTRUCTIONS.md](./BUILD_INSTRUCTIONS.md) for detailed deployment instructions.
