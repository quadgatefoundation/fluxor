#Add Website URL to Documentation

## Overview

Add the website URL `fluxor.io` to the documentation files to provide users with a link to the official website.

## Changes Required

### 1. DOCUMENTATION.md

Add website link in the Introduction section, right after the title or in the "What is Fluxor?" section:

```markdown
## Introduction

**Website**: [fluxor.io](https://fluxor.io)

Fluxor is a **reactive framework for building** scalable, event-driven applications in Go...
```



### 2. ARCHITECTURE.md

Add website link in the Overview section:

```markdown
## Overview

**Website**: [fluxor.io](https://fluxor.io)

Fluxor is a **reactive framework for building** applications in Go...
```



### 3. README.md

Add website link at the top of the README, typically after the title:

```markdown
# Fluxor

**Website**: [fluxor.io](https://fluxor.io)

A reactive framework for building...


```
